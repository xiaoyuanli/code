4. Conditional Panel

https://answers.splunk.com/answers/218623/how-to-hide-panels-with-no-results-from-a-dashboar.html

Note: the following example must have "as show" in the search tag value, because the only output will be values(show) but not show, then the condition will not be able to find this field.

<panel depends="$showPanel$">
<single>
<search>
    <query>index=resystem host=$host$ | head 1 | eval show=if(host=="releases.splunk.com", "Analyzing...", "Nothing") | stats values(show) as show</query>
    <earliest>-15m</earliest>
    <latest>now</latest>
<preview>
    <condition match="'result.show' == &quot;Analyzing...&quot;">
        <set token="test1">MATCH $result.show$ $result.host$</set>
        <set token="showPanel">true</set>
    </condition>
    <condition>
        <set token="test1">NOMATCH $result.show$ $result.host$</set>
        <unset token="showPanel"/>
        <set token="show">$result.show$</set>
    </condition>
</preview>
</search>

6. Drilldown

6.1 Code

Add drilldown in this way:

<drilldown target="App_Analysis_Window">
<link>/app/search/application_analysis?form.host=$host$&amp;form.app=$app$</link>
</drilldown>

Here, target is the window where the drilldown will be opened.

if set <drilldown target="_blank">, it will always open a new window.

Link needs the dashboard-ID to open: application_analysis, and the parameters: host and app.
